# CPA4Small Business – React Landing Page

Production-ready single-page React landing page for a Canadian accounting firm.

## Stack
- React + Vite
- Tailwind CSS
- Framer Motion
- Three.js via React Three Fiber
- Lucide React icons

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Structure
- `src/components` – reusable shared components
- `src/sections` – landing page sections
- `src/App.jsx` – page composition
- `src/main.css` – Tailwind utilities and global styles

## Notes
- Sticky navbar included
- Smooth scrolling enabled
- Responsive layout for mobile and desktop
- Hero includes animated Three.js particle background
- Pricing includes 4 yearly plans with Premium marked as Most Popular
